**Employee Management System (Java Desktop Application)**
**Project Description**

Employee Management System is a Java-based desktop application used to manage employee details such as adding, updating, viewing, and removing employee records using a graphical user interface.


**Tech Stack**

Java (Swing, AWT)

JDBC

MySQL Database

NetBeans IDE



**Features**

Login authentication

Add new employee details

Update employee information

View employee records

Remove employee details


**Output**

The application opens with a splash screen and login page. After login, users can manage employee records through a user-friendly desktop interface.
Database connectivity using MySQL
